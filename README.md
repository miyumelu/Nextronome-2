# neXtronome Aveatronic
An accompaniment system with the intention of creating a Korg-like accompaniment.

Aveatronic technology is used here to offer a modular design, simplifying maintenance and data transfer. This design is new, and many aspects still need to be adapted accordingly. This will happen during the course of development.

This is the newer version of neXtronome, based on .NET 10.0. It offers extended support, high compatibility, and secure functionality.

A .NET 6.0 version (Miyuse Univ) is still under development, as the features are being modified to provide better compatibility. The goal is compatibility from Windows 7 to 8.1.

For older versions, [neXtronome Legacy](https://github.com/miyumelu/nextronome-legacy) can be used.
